# 🧮 Custom Calculator (Python + CustomTkinter)

Современный калькулятор с графическим интерфейсом, написанный на Python. Проект создан с разделением логики и интерфейса для удобства поддержки.

## ✨ Особенности
*   **Современный UI**: Использование библиотеки `CustomTkinter` с поддержкой темной темы.
*   **Чистая архитектура**: Код разделен на три модуля (`main`, `ui`, `logic`).
*   **Безопасные вычисления**: Обработка ошибок (деление на ноль, некорректный ввод) через `try/except`.
*   **Адаптивная сетка**: Кнопки центрированы и выровнены с помощью метода `grid`.

## 🛠 Технологии
*   **Python 3.10+**
*   **CustomTkinter** (интерфейс)
*   **Math** (математические операции)

## 🚀 Как запустить

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com
   cd Calculator
   ```

2. **Создайте и активируйте виртуальное окружение:**
   ```bash
   python -m venv venv
   # Для Windows:
   venv\Scripts\activate
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Запустите приложение:**
   ```bash
   python main.py
   ```

## 📁 Структура проекта
*   `main.py` — точка входа, инициализация окна.
*   `ui.py` — создание всех визуальных элементов (кнопок, дисплея).
*   `logic.py` — математические функции и обработка нажатий.
